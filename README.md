# Task1
1.Titanic Dataset Analysis  (Day-1)

This project involves exploratory data analysis on the Titanic dataset to understand the structure, feature types, missing values, and overall suitability for machine learning tasks.

Dataset Information:

Dataset: Titanic – Machine Learning from Disaster (Kaggle)
Records: 418 rows
Features: 12
Target Variable: Survived (0 = Died, 1 = Survived)
Learning Type: Binary Classification

Task Objectives

Load and inspect the dataset

Identify data types (numerical, categorical, ordinal, binary)

Analyze missing values and feature quality

Assess dataset suitability for ML models

Deliverables

titanic.ipynb — dataset exploration and analysis

titanic_dataset.csv — dataset used for analysis

report.pdf — titanic/report.pdf
